# resource-optim


# 用途


集成视频/音频/图片压缩

系统：mac <br>
视频：mp4 <br>
音频：mp3 <br>
图片：png/jpg/jpeg <br>

#  大前提

由于压缩是调用ffmpeg工具 和 pngyu工具， 所以使用工具前先安装

###ffmpeg
https://ffmpeg.org/

### pngyu 
https://nukesaq88.github.io/Pngyu/
